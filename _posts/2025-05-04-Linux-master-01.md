리눅스 마스터 2급 기출문제 및 주요개념 정리 1
========


## 명령어
- XFS 파일 시스템을 생성: mkfs.xfs
- 셸의 정보를 확인: chsh -l
- yum을 통해 설치된 telnet-server 패키지를 제거: yum remove telnet-server
- 프로세스 ID(PID)로 우선순위를 변경: renice {nice값} -p {PID}
- 프로세스명 우선순위를 변경: nice -n {nice값} {명령어}
- 실시간 CPU 사용량 확인: top
- SSH 접속 시 사용할 포트 번호 지정: ssh -p
- 디스크 장치에 할당된 UUID를 확인: blkid


## 명령어와 주요 옵션

- tar 명령어 주요 옵션
  
  | 옵션 | 의미 | 설명 |
  |:----:|:----:|:----:|
  | `-c` | create | 새 tar 파일 생성 (압축 시작) |
  | `-x` | extract | tar 파일에서 압축 해제 |
  | `-v` | verbose | 진행 상황 출력 (파일 목록 보여줌) |
  | `-f` | file | 파일 이름 지정 (반드시 마지막에 사용) |
  | `-z` | gzip | gzip으로 압축/해제 (`.tar.gz`, `.tgz`) |
  | `-j` | bzip2 | bzip2로 압축/해제 (`.tar.bz2`) |
  | `-J` | xz | xz로 압축/해제 (`.tar.xz`) |
  | `-C` | directory | 압축 해제 시 대상 디렉토리 지정 |
  | `-r` | append | 기존 파일에 새 파일/디렉토리 추가 |
  | `-t` | list | 파일 내 포함된 파일/디렉토리 목록만 출력 |
  
- lpr 명령어 주요 옵션
  
  | 옵션 | 설명 |
  |:----:|:----:|
  |`-# <숫자>`|같은 문서를 여러 부 출력|
  |`-P <프린터명>`|특정 프린터를 지정하여 출력|
  |`-m`|인쇄가 끝난 후 메일로 사용자에게 통보|
  
- fdisk 명령어 주요 옵션
  
  |옵션|설명|
  |:---:|:---:|
  |`-l`|list:시스템의 모든 디스크와 파티션 정보를 나열|
  |`-v`|version:fdisk의 버전 정보를 출력|
  |`-t`|type:파일 시스템의 타입을 지정|
  |`-s`|size:특정 파티션의 크기를 블록 단위로 출력|

- netstat 명령어 주요 옵션
  
  |옵션|설명|
  |:---:|:---:|
  |`-r`|route/라우팅 테이블 출력|
  |`-t`|tcp 연결만 출력|
  |`-m`|memory/네트워크와 관련된 메모리 사용 정보를 출력|
  |`-n`|numeric/호스트 이름 대신 IP주소를 숫자 형태로 표시|

- rpm 명령어 주요 옵션
  
  |옵션|설명|
  |:---:|:---:|
  |`-q`|패키지 정보 조회|
  |`-qa`|설치된 전체 패키지 목록 조회|
  |`-qi`|패키지의 자세한 정보 확인|
  |`-ql`|패키지가 설치한 파일 목록 확인|
  |`-qc`|패키지의 설정 파일 목록 확인|
  |`-qf`|특정 파일이 어떤 패키지에 속하는지 확인|
  |`-e --nodes`|의존성 무시하고 제거|


## 환경변수
- USER: 현재 로그인한 사용자 이름
- HOME: 사용자의 홈 디렉토리 경로
- SHELL: 기본으로 사용하는 셸 프로그램 (사용자의 로그인 셸이 기록)
- PWD: 현재 작업 중인 디렉토리
- PS1: 기본 프롬프트 (ex: $, #)
- PS2: 줄이 계속될 때 나오는 2차 프롬프트 (역슬래시 사용)
- PS3: select 명령에서 선택지 앞에서 나오는 프롬프트
- PS4: set -x 디버깅 시 앞에서 붙는 표시

## OSI 7계층 

| OSI 계층 | 계층 이름 (한글) | 대표 기능 | 대표 장비 예시 |
|:---:|:---:|:---:|:---:|
| 7 | 응용 계층 (Application)   | 사용자 인터페이스, 웹 서비스 | **L7 스위치**, 애플리케이션 게이트웨이 |
| 6 | 표현 계층 (Presentation)  | 암호화, 압축, 변환 | (일반 장비 없음, 소프트웨어 처리) |
| 5 | 세션 계층 (Session)       | 세션 설정 및 관리 | (일반 장비 없음, 소프트웨어 처리) |
| 4 | 전송 계층 (Transport)     | TCP/UDP 포트 제어 | 고급 방화벽 (포트 기반 필터링) |
| 3 | 네트워크 계층 (Network)     | IP 주소 기반 라우팅 | **라우터**, L3 스위치, 방화벽 |
| 2 | 데이터 링크 계층 (Data Link) | MAC 주소 기반 스위칭 | **스위치(L2)**, 브리지 |
| 1 | 물리 계층 (Physical)      | 전기 신호, 케이블 전송 | **허브**, 리피터, 케이블 |

- MAC 주소 변환은 ARP

## 주소 계산
- 네트워크 주소: IP 주소 OR (NOT 서브넷 마스크)
- 브로드캐스트 주소: IP 주소 AND 서브넷 마스크
- 게이트웨이 주소: 네트워크 주소 ~ 브로드캐스트 주소 사이 유효 주소 (보통 첫번째 유효 주소)
  
  | 항목 | 값 |
  |:---:|:---:|
  | **IP 주소** | 192.168.5.66 |
  | **서브넷 마스크** | 255.255.255.192 (/26) |
  | **네트워크 주소** | 192.168.5.64 |
  | **브로드캐스트 주소** | 192.168.5.127 |
  | **게이트웨이 주소 (보통)** | 192.168.5.65 |
